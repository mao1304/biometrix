¿Qué es BiometrixUCC?

BiometrixUCC utiliza Django Rest Framework para el backend, que es la parte encargada de gestionar la lógica de la aplicación. Django Rest Framework facilita la creación de APIs para la comunicación entre el frontend y el backend. Por otro lado, el frontend está desarrollado con React Native, lo que significa que la aplicación es multiplataforma y puede ejecutarse en dispositivos iOS y Android. La autenticación se realiza mediante la validación de la huella dactilar del celular, lo que proporciona un nivel adicional de seguridad. En resumen, BiometrixUCC ofrece una solución integral para el control de asistencia de docentes, aprovechando tecnologías modernas y garantizando la autenticación a través de la biometría.
